github 上的项目

前端

  • tholman/github-corners: 漂亮的 GitHub 页角. 基于 SVG. 取代了多年的fork me on github(在github上给我来一叉子)
  • rainyday.js 给你的图片加上下雨的效果 (想起了friends里joey家里那个自动下雨的窗)
  • css easing 使用css3来使动画有淡入淡出的效果
  • sigma.js 又一个基础的绘图库
  • plot.ly 一个商业的图表公司的js开源库, 还有r,matlab等多个版本
  • d3 大名鼎鼎的d3

看似和jQuery类似，实则可以绘制SVG的强大工具 (quote from swaylq)

  • vivus svg动画效果

两者配合起来什么复杂的动画都可以做了。(quote from swaylq)

后端

  • beautified php error display 好像laravel4的错误显示就是这个样子的, 另外phalcon的报错也很漂亮
  • Laravel5 LogViewer 就是咩汉化的那个日志浏览工具
  • Laravel LogViewer 同上, 只是没有那么好看
  • Laravel DebugBar 显示数据库查询etc (对使用restful api的并没什么用, 其实应该开个pr, 然后hook $http, 获取ajax请求的资源消耗情况)
  • fxa-content-server 火狐的账户系统
  • atmosphere java的实时应用服务器, 支持轮询和websocket
  • shiro java的权限验证系统
